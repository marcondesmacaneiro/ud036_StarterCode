#  Movie Trailer by Marcondes

## Descrição
* Esse projeto consite em um sistema server-side para divulgação de meus filmes favoritos.
São apresentados algumas características dos filmes e também o poster de divulgação do filme e um trailer do Youtube

## Requerimentos
* Python 2.7.x

## Instalação
* Faça download ou clone do repositório em seu computador
* Extraia o repositório caso o Download tenha sido feito com um arquivo .zip
* Rode o arquivo my_movies.py

## Adicionar ou Editar Filmes
* Abra o arquivo my_movies.py
* Crie um novo objeto conforma documentação apresentada
* Adicione esse novo objeto ao array de filmes que está no final do arquivo e conforme documentação apresentada
* Rode o arquivo my_movies.py
